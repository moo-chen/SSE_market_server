# 软工集市奖励机制

## 等级机制

| 积分区间  | 等级头衔 |
| --------- | -------- |
| 小于100   | 菜鸟     |
| 100-300   | 大虾     |
| 300-600   | 码农     |
| 600-1000  | 程序猿   |
| 1000-2000 | 工程师   |
| 2000-3000 | 大牛     |
| 3000-4000 | 专家     |
| 4000-5000 | 大神     |
| 5000以上  | 祖师爷   |

## 积分机制

| 行为         | 积分奖励 |
| ------------ | -------- |
| 发帖         | +10      |
| 评论/回复    | +5       |
| 被评论/回复  | +2       |
| 被收藏       | +3       |
| 被点赞       | +1       |
| 举报他人成功 | +20      |
| 被举报成功   | -20      |
